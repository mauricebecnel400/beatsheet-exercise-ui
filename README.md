# beatsheet-exercise-ui
## Installation

1) start docker container from https://github.com/fmatar/beatsheet-exercise 

2) clone down repo

3) open `.env.local` and update `VITE_BEAT_SHEET_API_HOST` to whatever port the container is running on

4) run `npm install`

5) run `npm run dev`
